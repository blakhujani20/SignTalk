About SignTalk

SignTalk is an AI-powered web application designed to bridge communication between the deaf and non-deaf. It translates real-time sign language into readable text and voice using machine learning and computer vision.

Built using Flask, TensorFlow, and MediaPipe, the app offers:

  • ✓ Real-time ASL hand detection
  • ✓ Live video call with automatic sign recognition
  • ✓ Text-to-speech integration for hearing participants
  • ✓ Multi-device support via SocketIO and WebRTC

Project Information

This project was developed by Bhavesh Lakhujani, B.E. student in AI & Data Science at MBM University, Jodhpur.
